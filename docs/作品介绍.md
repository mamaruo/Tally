# Tally 记账 App 作品介绍

## 1. 作品简介
Tally 是一款基于 Android Jetpack Compose 开发的个人记账 App。它支持离线使用，数据保存在本地。主要功能包括记录收入和支出、查看每月收支统计、以及管理交易分类。

## 2. 功能展示

### 2.1 主页
主页顶部显示当月的总支出和结余。下方是按日期分组的交易列表，方便查看每天的消费情况。
【主页截图】
*技术实现：使用 `LazyColumn` 构建列表，根据日期对交易数据进行分组展示。*

### 2.2 记账
点击主页的悬浮按钮 (FAB) 即可记账。页面支持切换收入/支出，输入金额，选择日期和分类。
【新增交易截图】
*技术实现：使用 `LargeFloatingActionButton` 跳转，`OutlinedTextField` 输入金额，`DatePicker` 选择日期。*

### 2.3 分类管理
用户可以查看现有的分类，也可以添加新的分类（自定义名称和图标）。
【分类管理截图】
*技术实现：图标使用 `androidx.compose.material.icons` 库，通过 `IconRegistry` 管理图标资源。*

## 3. 技术特点
- **纯 Compose UI**: 界面完全使用 Jetpack Compose 构建，遵循 Material Design 3 规范。
- **本地存储**: 使用 Room 数据库存储交易和分类数据，支持离线运行。
- **架构设计**: 采用 MVVM 架构和 Hilt 依赖注入，代码结构清晰。
